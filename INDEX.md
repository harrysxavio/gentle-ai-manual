# Índice pedagógico

---

## Cómo usar este manual

Este manual está organizado en 20 módulos que van desde fundamentos tecnológicos hasta construcción de productos reales.

Cada módulo indica:
- **Nivel**: 1 (principiante), 2 (intermedio), 3 (avanzado)
- **Tiempo estimado**: minutos de lectura
- **Prerrequisitos**: qué necesitás saber antes
- **Competencias**: qué vas a aprender
