# Skill Evaluation Rubric

Required behavior:

- loads the skill and supporting references;
- inspects curriculum and neighboring pages;
- builds a source matrix;
- refuses invented commands;
- separates conceptual and volatile claims;
- uses progressive audience depth;
- uses a continuous example;
- includes trade-offs and verification;
- updates route, glossary and tests;
- applies web/mobile constraints;
- handles screenshot provenance.

Each skill-enabled scenario must score at least 90/100 and contain no hard failure.
